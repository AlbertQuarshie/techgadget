# TechGadget Comparison

This project showcases a comparison page for tech gadgets using only HTML and CSS.

## Methods Used

### A. Layout and Structure
1. Created a main container to hold all content.
2. Designed two product articles: Standard Model and Pro Model.
3. Added a feature list using an unordered list and a footer for social links.

### B. Typography and Styling
1. Centered all text and applied a bisque background color.
2. Styled headings and paragraphs for readability.
3. Used padding and borders to differentiate sections.

### C. Buttons
1. Styled "Buy Now" buttons with background colors.
2. Added hover effect to change the button color for interactivity.

### D. Lists and Features
1. Styled the feature list with alternating background colors using `nth-child(even)`.
2. Removed default list bullets and adjusted padding.

### E. Footer
1. Added social media links with icons (Instagram and Facebook).
2. Styled icons with fixed height for consistency.
